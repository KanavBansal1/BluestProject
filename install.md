# Installation Guide

Follow these steps to set up the Multi-Modal Synthetic Data Generator on your local machine.

## Prerequisites
Ensure you have the following installed on your system:
- Python 3.x

## Step 1: Install Dependencies
Open your terminal and run the following command to install the required Python packages:

```bash
pip install flask numpy pillow
```

## Step 2: Run the Application
Once the dependencies are installed, you can start the Flask web server by running:

```bash
python app.py
```

## Step 3: Access the Application
Open your preferred web browser and navigate to the following URL to start generating synthetic data:

[http://localhost:5000](http://localhost:5000)
